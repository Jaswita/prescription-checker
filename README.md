#  Secure Prescription Verification & Tracking System\n\nA robust, enterprise-grade system for secure prescription management, verification, and controlled substance tracking. Built with high-security standards, audit compliance, and a modern full-stack architecture.\n\n---\n\n##  Key Features\n\n###  Prescription Lifecycle Management\n- **Secure Creation**: Doctors can issue encrypted prescriptions using AES-256 and RSA signatures.\n- **Real-time Verification**: Pharmacies can instantly verify prescription authenticity via QR codes.\n- **Secure Dispensing**: Integrated tracking for medicine dispensing with automatic inventory updates.\n- **Controlled Substance Tracking**: Special protocols for high-risk medications to prevent abuse.\n\n###  Advanced Security Suite\n- **Multi-Layer Encryption**: Dual encryption using AES for data and RSA for digital signatures.\n- **Two-Factor Authentication (2FA)**: OTP-based secure login via Email (Nodemailer).\n- **Brute Force Protection**: Account lockout after multiple failed attempts and rate limiting.\n- **Session Management**: Secure JWT-based authentication with auto-logout features.\n\n###  Comprehensive Audit Logging\n- **18 Event Types Tracked**: Full coverage from logins to prescription dispensing.\n- **Dual Storage Strategy**: \n  - **Database**: Indexed MongoDB records for fast admin queries.\n  - **Flat Files**: Human-readable .log files for forensic evidence.\n- **Immutable Records**: Logs are protected against deletion and modification.\n\n###  Admin & Pharmacy Dashboards\n- **Security Monitoring**: Track failed access attempts and account lockouts.\n- **Analytics & Reporting**: Visual statistics on system activity.\n- **Data Export**: CSV export functionality for regulatory compliance and audits.\n\n---\n\n##  Technology Stack\n\n### Frontend\n- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/) & [Vite](https://vitejs.dev/)\n- **UI Library**: [React 19](https://reactjs.org/)\n- **Styling**: [Tailwind CSS](https://tailwindcss.com/) with [Radix UI](https://www.radix-ui.com/) components\n- **Icons & Visualization**: [Lucide React](https://lucide.dev/) & [Recharts](https://recharts.org/)\n- **State & Forms**: [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)\n\n### Backend\n- **Runtime**: [Node.js](https://nodejs.org/)\n- **Framework**: [Express.js](https://expressjs.com/)\n- **Database**: [MongoDB](https://www.mongodb.com/) with [Mongoose ODM](https://mongoosejs.com/)\n- **Security**: [Helmet](https://helmetjs.github.io/), [Bcrypt.js](https://github.com/dcodeIO/bcrypt.js), [jsonwebtoken](https://github.com/auth0/node-jsonwebtoken)\n\n### Utilities\n- **Scanning**: [Html5-Qrcode](https://github.com/mebjas/html5-qrcode)\n- **Email**: [Nodemailer](https://nodemailer.com/)\n- **QR Generation**: [qrcode](https://github.com/soldair/node-qrcode)\n\n---\n\n##  Getting Started\n\n### Prerequisites\n- Node.js (v18+)\n- MongoDB (Atlas or Local)\n\n### Setup\n1. **Clone the repository**\n2. **Install dependencies**: npm install (root, frontend, and backend)\n3.
